# Hey, I'm Vivek

I'm a full-stack software engineer who likes building things that actually ship: clean frontends, reliable backends, mobile apps, developer tools, and the occasional fix buried three folders deep in someone else's monorepo.

I work across React, React Native, Expo, Node.js, TypeScript, native/mobile tooling, and AI-agent systems. I enjoy turning vague bugs into small, mergeable PRs, making CLIs behave like humans expect, and improving docs so the next developer does not have to perform archaeology with six tabs open.

## Things I do

- Build full-stack apps from UI to database, and all the odd glue in between
- Ship React, React Native, and Expo work that survives real devices
- Fix developer tooling, CLIs, flaky workflows, and "why is this hardcoded to 8081?" moments
- Contribute to large open-source repos without moving the furniture around
- Write docs that answer the question before someone rage-searches it
- Debug with optimism, logs, and just enough suspicion
- Treat TypeScript like a seatbelt, not a personality
